# supervision-site

Готовый сайт для загрузки на GitHub и публикации через Vercel.

## Что внутри
- index.html
- src/App.jsx
- src/main.jsx
- package.json

## Как опубликовать
1. Распакуйте архив.
2. На GitHub откройте репозиторий.
3. Нажмите Add file → Upload files.
4. Перетащите ВСЕ файлы и папки из архива.
5. Нажмите Commit changes.
6. Зайдите на vercel.com и войдите через GitHub.
7. Нажмите Add New Project.
8. Выберите этот репозиторий.
9. Нажмите Deploy.
